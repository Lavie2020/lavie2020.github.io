准备开始 Python 学习

发表于2021-07-11|更新于2021-07-15|Python学习
|阅读量:

实习工作的第一天

  • Python技能方面

    复习了一下xpath的语法和使用方法，尝试对工作网站进行了标签定位，熟练了一点,想要吐槽的是凯棋在面试的时候说以后日常从事的工作和我当时说得爬取网页差不多，结果一个是用最基本的request.get就完事了，一个是用scrapy框架，不过毕竟是大项目嘛，也十分合理，我现在也想快速上手，爬一个网站估计特别有成就感

明天早上早点过去，让泽塔给我开个小灶，恶补下，拼搏三天，我要爬网站做外包！

  • 人际方面

    认识了和我一起来实习的两位同事，不过跟派仔不太熟悉，感觉派仔应该是个精英，估计用不了多久就能掌握工作技能，至于猛男，那是真的猛男，不过猛男没用python基础，应该挺痛苦的。
    嘿嘿，还认识了泽塔，听到他花名的时候我就在想，这是不是泽塔奥特曼的泽塔，就问了一下，没想到还真是，身为半个假面骑士厨和幼年时期奥特曼厨，也算是和他有共同语言了。不得不说，泽塔人真得贼好，工作上很多不懂的问题都是他教我的，今天下班还和他聊了会，挺投缘的，可能这就是缘分吧！

  • 未来展望

    希望能在短时间内快速掌握scrapy，至少能运用到工作上，摸鱼实在是太痛苦了，盯着电脑看一堆scrapy的学习视频，但是内容和工作上要用的完全不是一个量级的，想从实战起步，快点积累经验。
